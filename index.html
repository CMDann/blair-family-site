About This Site

Daniel Blair 2017

This is a simplen website which serves the purpose of gathering the geneological information for the blair family clan in Winnipeg Manitoba, Canada. This is a continuation of the work put in by Peggy Blair and digitized by Daniel Blair. This data is open source and is intended for you to view and contribute. We want to make sure that this research is as updated as possible. You can contribute through our geneology site or through the GitHub repository.
